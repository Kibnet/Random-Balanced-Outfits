=================
Генератор нарядов
=================
Приложение предназначено для автоматизации процесса распредления нарядов и дежурств между подразделениями наиболее честным образом.
Каждому подразделению назначаются наряды пропорционально количеству человек. 
При распределении происходит балансировка нагрузки (часов на человека), для уменьшения разброса отклонений от нормы нагрузки.
Также алгоритмом предусмотрино перемешивание нарядов таким образом, чтобы по возможности исключались случаи постановки в наряд нескольких человек из одного подразделения.

Всё взаимодействие с программой разделено на 4 этапа.
Каждому этапу соответствует раскрывающийся блок программы

1. Выставление периодов дат (блок "Даты")
Выберите в календариках период графика, дни усилений и праздничные дни.

2. Настройка списка нарядов (блок "Наряды")
Сформируйте список дежурств и нарядов которые вы хотите распределить.
Для добавления нового наряда нажмите кнопку с изображением "+".
Во всплывающем окне введите название наряда, его длительность, в какие дни в него заступают и является ли он усиленным.
Для редактирования наряда можно открыть его двойным щелчком мыши.

3. Настройка списка подразделений (блок "Подразделения")
Сформируйте список подразделений, которых надо назначить в наряды и дежурства.
Для добавления нового подразделения нажмите кнопку с изображением "+".
Во всплывающем окне введите название подразделения, количество человек и наряды в которые это подразделение можно назначать.
Для редактирования подразделения можно открыть его двойным щелчком мыши.
При нажатии кнопки "Сохранить Статистику..." можно выбрать путь для сохранения и список полей таблицы Подразделения.
Статистика сохраняется в формате *.doc(Microsoft Office Word).

4. Распределение нарядов между подразделениями (блок "График")
В этом блоке генируется график нарядов исходя из заполненых данных блоков "Даты" и "Наряды". 
Заполняется этот график данными из блока "Подразделения".
Есть возможность заблокировать определённые наряды в определённые дни, чтобы они не меняли своего значения и соответсвенно не участвовали в расчёте нагрузки.
Кнопка "Распределить наряды" запускает алгоритм случайной расстановки подразделений в ячейки графика и их перестановки.
Алгоритм прогоняется 100 раз.
По завершению работы алгоритма выбирается самый сбалансированный вариант.
При нажатии кнопки "Сохранить График..." можно выбрать путь для сохранения получившейся таблицы нарядов.
График сохраняется в формате *.doc(Microsoft Office Word).
